## üé® After Approval: Integration Guide

### **Step 1: Get Your Affiliate Links**

**In Partner Dashboard:**
1. Go to "Marketing Tools"
2. Choose link type:
   - **Deep Links** - Link to specific hotels
   - **Search Links** - Link to city/destination search
   - **Banner Ads** - Display ads (optional)

### **Step 2: Create Affiliate Links**

#### **City Search Link:**
```
https://www.booking.com/city/[country]/[city].html?aid=[YOUR_AID]
```

Example for Paris:
```
https://www.booking.com/city/fr/paris.html?aid=YOUR_AID
```

#### **Hotel Search Link:**
```
https://www.booking.com/searchresults.html?city=[ID]&aid=[YOUR_AID]
```
